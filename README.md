## para executar siga as seguintes instruções:

- Clona o ropositório

  ```bash
  https://github.com/Jefersonwork/bookings.git
  ```

- Instalar as dependências do projeto
 
   * yarn
  ```bash
  yarn install
  ```

  * npm
  ```bash
  npm install
  ```
- Rodar o projeto
 
   * yarn
  ```bash
  yarn start
  ```

  * npm
  ```bash
  npm run start
  ```
